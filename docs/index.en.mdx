**Jangada AI** 🛶 is a thin layer over the official LLM SDKs
(Anthropic, OpenAI, Groq, Gemini). The goal is to **swap provider/model/api_key
without changing the rest of your code**, with `{{ }}` templates, chained flows,
structured output (Pydantic), vision, audio, async, and error-based fallback.

## Installation

```bash
pip install "jangada-ai[anthropic]"
```
