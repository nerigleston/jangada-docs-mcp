A **Jangada AI** 🛶 é uma camada fina sobre os SDKs oficiais de LLM
(Anthropic, OpenAI, Groq, Gemini). O objetivo é **trocar provider/model/api_key
sem mudar o resto do código**, com templates `{{ }}`, fluxos encadeados,
structured output (Pydantic), vision, áudio, async e fallback por erro.

## Instalação

```bash
pip install "jangada-ai[anthropic]"
```
